# プロダクト概要

medi-test は、medimo 向けにテストシナリオを保存・再利用し、リリース単位で実行記録を管理するサービスです。テストの「何を」「誰が」「いつ」「どの結果で」実施したかを追跡できます。

## サービス名

- medi-test

## 対象ユーザーと権限

- 管理者
- 実行者
- 閲覧者
- 承認者
- 役割は追加可能

## 主要機能

- シナリオの作成、更新、アーカイブ
- リリース単位のテストラン作成と担当割り当て
- 実行結果の記録と証跡保存（DB）
- 重要度別の合格条件による完了判定
- GitHub PR のリリースタグ検出
- Linear からの変更内容取得
- テスト範囲の自動提案
- 複数プロジェクトと環境の管理

## 完了条件

- 重要度別の合格条件に基づいて判定

## 実行結果ステータス

- 成功
- 失敗
- 未実施

## 必須・任意

- テストラン内で各シナリオを必須または任意として設定可能

## 証跡

- 本ツールで入力した内容を証跡として保存
- 保存先は DB

## 外部連携

- GitHub PR のリリースタグを参照
- PR と紐づいた Linear の情報を取得

## 将来拡張

- 自動テスト結果の取り込みや連携に対応する余地を保持
